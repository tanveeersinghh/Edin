# HeyEdin

Step 1: Clone the repository <br />
$ git clone https://github.com/tanveeersinghh/Edin.git

Step 2: Initialise it using npm init <br />
$ npm init -y

Step 3: Make it global <br />
$ npm link

# How to Use?
$ HeyEdin {FunctionName} {optional argument}

$ HeyEdin help <br />
